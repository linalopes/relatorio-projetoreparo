Como navegar

Use as setas do teclado ou clique sobre os botões de navegação no canto direito inferior da tela.


Visão Geral

Pressione ESC para o modo slide overview.


Zoom

Mantenha a tecla ALT pressionada e clique em qualquer elemento para aumentar o zoom. ALT + clique em qualquer lugar para diminuir o zoom.


Exportar para PDF

Você pode exportar este relatório para PDF aqui.

Modo Apresentação com Notas

Há uma versão de apresentação. Inclui um temporizador, pré-visualização do próximo slide, bem como suas anotações de orador.

Pressione S para experimentar.


Pausa

Pressione B ou . no seu teclado para pausar a apresentação.
